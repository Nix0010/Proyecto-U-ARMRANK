## Proyectos destacados

### ARMRANK

Sistema web full-stack para gestion de torneos, participantes, categorias, brackets, rankings y resultados.

**Problema que resuelve:** muchos eventos se administran con hojas de calculo, mensajes y procesos manuales. ARMRANK centraliza la informacion y reduce errores al organizar competencias.

**Incluye:**

- Frontend con React, TypeScript y Tailwind CSS.
- Backend con Express, Prisma y PostgreSQL.
- Autenticacion con JWT y Google OAuth.
- Brackets de eliminacion simple, doble eliminacion, round robin y vendetta.
- Gestion de participantes, categorias, rankings y podios.
- Enlaces publicos para compartir torneos.
- Exportacion de resultados a PDF.

Repositorio: [Proyecto-U-ARMRANK](https://github.com/Nix0010/Proyecto-U-ARMRANK)

### ARMPRO-F

Proyecto frontend en TypeScript orientado a construir interfaces web modernas. Puede funcionar como base para paginas, paneles o aplicaciones de negocio.

Repositorio: [ARMPRO-F](https://github.com/Nix0010/ARMPRO-F)

### BaseDeDatos

Sistema de gestion academica con enfoque en tareas, horarios, calificaciones y recordatorios. Muestra modelado de informacion y organizacion de datos para una necesidad concreta.

Repositorio: [BaseDeDatos](https://github.com/Nix0010/BaseDeDatos)
